============================================================
===================== 菜单, 快捷键说明 =====================
============================================================


------------------------------------------------------------
------------ 支持秒表, 时钟, 时间戳3种样式切换 -------------
------------------------------------------------------------
Style - Timer           # 样式: 秒表      >> 快捷键: Ctrl + t
Style - Clock           # 样式: 时钟      >> 快捷键: Ctrl + k
Style - Timestamp       # 样式: 时间戳    >> 快捷键: Ctrl + m


------------------------------------------------------------
---------------- 支持增大, 减小, 重置字号 ------------------
------------------------------------------------------------
Font size - Increase    # 字号: 增大      >> 快捷键: Ctrl + =
Font size - Decreate    # 字号: 减小      >> 快捷键: Ctrl + -
Font size - Default     # 字号: 50        >> 快捷键: Ctrl + d


------------------------------------------------------------
--------------------- 支持设置刷新帧率 ---------------------
------------------------------------------------------------
Frame rate - 240        # 帧率: 240       >> 快捷键: Ctrl + 4
Frame rate - 120        # 帧率: 120       >> 快捷键: Ctrl + 2
Frame rate - 90         # 帧率: 90        >> 快捷键: Ctrl + 9
Frame rate - 60         # 帧率: 60        >> 快捷键: Ctrl + 6
Frame rate - 50         # 帧率: 50        >> 快捷键: 无
Frame rate - 40         # 帧率: 40        >> 快捷键: 无
Frame rate - 30         # 帧率: 30        >> 快捷键: Ctrl + 3
Frame rate - 20         # 帧率: 20        >> 快捷键: 无
Frame rate - 15         # 帧率: 15        >> 快捷键: Ctrl + 5
Frame rate - 10         # 帧率: 10        >> 快捷键: Ctrl + 1
Frame rate - 5          # 帧率: 5         >> 快捷键: 无
Frame rate - 1          # 帧率: 1         >> 快捷键: 无


------------------------------------------------------------
--------------------- 开始, 暂停, 清空 ---------------------
------------------------------------------------------------
Action - Play           # 操作: 开始      >> 快捷键: Ctrl + p
Action - Pause          # 操作: 暂停      >> 快捷键: Ctrl + p
Action - Clear          # 操作: 清空      >> 快捷键: Ctrl + l
